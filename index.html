BBC Charts UI Library

A multi-functional JS/CSS chart library with suitable and accessible fallbacks. With lots of configurable options and different views. An image generator is also available to help with the creation of image charts for when the library cannot be used.

Principles

  • Data is interchangeable between charts
  • Minimal JS needed for chart to function
  • Add as much or as little html as you want to create the chart (JS functions can populate the bits missing)
  • CSS Vars can be used customise the chart (Sass vars can also be used)
  • Interaction to show/hide datasets
  • Charts update when the data is changed; allowing for live charts
